MTMENU
------

PyMT application for selecting and executing multi-touch applications. Belongs to the WallManager platform, designed to run on the SenseWall multi-touch device in the Departament of Informatics Engineering of the University of Coimbra.


REQUIREMENTS
------------

- Windows only (tested in XP, Vista, Windows 7)
- Python 2.6
- PyMT 0.4 (and respective dependencies)
- TUIO Server, configured to the port 6000, or a TUIO Simulator (http://tuio.org/?software)
- Win32 API para Python

Application Libraries (not required to run the platform):
 - MT4J
 - OpenFrameworks


EXECUTION
---------

Run the platform:
	python launcher.py
