3. What do I need to compile?

Compiling Applications

All you need to do, is just your normal compilation of the application for your chosen language, and create the launchable batch file.
(You can see details about each supported library at the section 2. Libraries Supported.)


So, for example, if your application was written in:

  • OpenFrameWorks: just compile it into an *.exe
  • MT4j: compile it into a *.jar
  • Python (and other scripting languages): there is no need to compile your source code to execute your applications. Although not recommended and unnecessary, you can compile your python applications for whatever reason. Some information about python compilation.


Just remember to create your batch file (*.bat), as explained at the section 6. Format of the File to Upload, to make it launchable!
